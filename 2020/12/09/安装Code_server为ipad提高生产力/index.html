安装Code_server为ipad提高生产力

2020-12-09
Word count: 947   |   Reading time≈ 4 min

标题

在网上看到有人拿ipad来手写代码，发现是个很不错的东西，不用抱着笔记本来学习代码了，毕竟笔记本还是不能做到随时随地拿出来，论便携还是平板厉害（扯远了

1.下载code-server

Code-server的Github地址是:https://github.com/cdr/code-server
点击Release可以看到以往版本的Code-server
这里我安装的是3.7.4版本
复制对应版本的压缩包的下载链接

2.解压缩这个安装包

3.打开code-server

这个时候在每个设备的浏览器输入{服务器IP:code-server端口}，输入密码就能打开在自己服务器的VScode

4.让服务器在后台运行code-server

前面打开的code-server会在关闭ssh客户端时被一并关闭，想要随时随地打代码总不可能24小时开着ssh客户端吧。所以我们要去服务器的云控制台里

5.安装Python插件

由于我等了很久的python在线安装都好像没反应，所以转.vsix方式离线安装vscode的python插件
在github找到python插件的.VSIX文件，用wget下载到本地。
一样是右键需要下载的包复制下载链接

然后再code-server的扩展中，用.vsix安装的方式把下载的包安装进去就好了

6.安装python3.7

由于linux里自带的是Python2.7，一开始缩进错误折磨了好久，后来安装了Python3.7之后就没有这方面的问题了

首先 安装依赖库

然后是 下载Python3.7源码

接着是 执行配置文件、编译、编译安装

安装完成没有提示错误就证明安装成功了

最后是 建立软连接
